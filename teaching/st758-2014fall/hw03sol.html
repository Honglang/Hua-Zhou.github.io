For (a) and (b), check the definition of Moore-Penrose inverse directly.

ginv.courrier <- function(M, method = "chol", b = NULL) {
    
  # check dimension
  if (is.matrix(M)) {
    m <- dim(M)[1]
    n <- dim(M)[2]
    # let's work on a tall matrix
    if (m < n) {
      M <- t(M)
      transM <- TRUE
      } else {
        transM <- FALSE
        }
    } else {
      stop("M must be a matrix")
    }
  if (!is.null(b) && dim(b)[1] != m)
    stop("b should have same number of rows as M")
  
  if (method == "chol") {
    # Cholesky decomposition (with symmetric pivoting) of Grammian matrix
    mtm.lt <- chol(crossprod(M), pivot = TRUE)
    pivot <- attr(mtm.lt, "pivot")
    rank <- attr(mtm.lt, "rank")
    mtm.lt <- mtm.lt[1:rank, ]
    } else if (method == "qr") {
      # QR decomposition of M
      mqr <- qr(M)
      pivot <- mqr$pivot
      rank <- mqr$rank
      mtm.lt <- qr.R(mqr)[1:rank, ]
    }
  
  # prepare right hand side(s)
  # Note on pivot: 
  # M[, pivot] == M %*% t(P), M[, order(pivot)] == M %*% P
  # M[pivot, ] == P %*% M, M[order(pivot), ] == t(P) %*% M
  if (is.null(b)) {
    ginvb <- tcrossprod(mtm.lt, M[, pivot])
    } else {
      ginvb <- mtm.lt %*% crossprod(M[, pivot], b)
    }
  # compute the "matrix inverse" in middle by Cholesky
  if (method == "chol") {
    # Cholesky decomposition of the Grammian matrix t(L) %*% L
    ltl.cholR <- chol(tcrossprod(mtm.lt))
    } else if (method == "qr") {
      ltl.cholR <- qr(t(mtm.lt))$qr
  }
  # two rounds of forward/backward solve to finish
  ginvb <- forwardsolve(ltl.cholR, ginvb, upper.tri = TRUE, transpose = TRUE)
  ginvb <- backsolve(ltl.cholR, ginvb)
  ginvb <- forwardsolve(ltl.cholR, ginvb, upper.tri = TRUE, transpose = TRUE)
  ginvb <- backsolve(ltl.cholR, ginvb)
  ginvb <- crossprod(mtm.lt, ginvb)[order(pivot), ]
  
  # output
  if (transM) {
    return(t(ginvb))
  } else {
    return(ginvb)
    }
  }

Use QR for linear regression. Applying qr function to \(X\) yields \(XP = QR\). That is \(P^T X^T X P = R^T R\). Normal equation \(X^T X \beta = X^T y\) is equivalent to \(P^T X^T X P (P^T \beta) = P^T X^T y\). Thus we first solve for \(P^T \beta\) (by 2 triangular solves) and then permute back to obtain \(\hat \beta\).

To use the sweep operator for linear regression, see the demo code at http://hua-zhou.github.io/teaching/st758-2014fall/sweep.html.

  1. R uses Householder QR for the linear regression.

  2. Setting the gradient (the vector of derivatives) to \(\mathbf{0}\), we obtain the stationarity condition
    \[ X^T (y - X \beta) + \lambda \beta = \mathbf{0}. \] Since the objective function is convex, any stationary point is a global minimum. Thus the ridge solution is given by
    \[ \hat \beta = (X^T X + \lambda I)^{-1} X^T y. \] Uniqueness follows from the non-singularity of the matrix \(X^T X + \lambda I\).

  3. Writing the ridge criterion as
    \[ \frac 12 \left \| \begin{pmatrix} \mathbf{y} \\ \mathbf{0}_p \end{pmatrix} - \begin{pmatrix} \mathbf{X} \\ \sqrt{\lambda} \mathbf{I}_p \end{pmatrix} \mathbf{\beta} \right\|_2^2, \] we see it is equivalent to an augmented linear model with \(n+p\) observations. Below we solve the ridge regression at a sequence of \(\lambda\) values using Cholesky decomposition. We see \(\|\beta(\lambda)\|_2\) decreases as a function of \(\lambda\).

library(ggplot2)
lambdas = seq(0, 100, 5)
betas = matrix(0, 7, length(lambdas))
for (k in 1:length(lambdas)) {
  M <- yXXy
  # add ridge to diagonal
  diag(M)[1:7] <- diag(M)[1:7] + lambdas[k]
  cholR <- chol(M)
  betas[, k] <- backsolve(cholR[1:7, 1:7], cholR[1:7, 8])
}
